Language Syntax

Clean. Expressive.
Built for the Metaverse.

KursarScript (KSPL) draws inspiration from modern scripting languages while adding first-class primitives for virtual economies — so you spend less time wiring and more time building.

  • ✓ Familiar syntax — let, fn, class, for, while feel instantly natural.
  • ✓ VR built-ins — Avatar, VirtuCard, VirtualTerminal, and VirtualItem are first-class.
  • ✓ No inheritance — a simplified, flat object model that's easier to reason about in complex VR systems.
  • ✓ Multi-line expressions — lists, dicts, and arithmetic can span multiple lines cleanly.
economy.kursar
// Create VR avatars with VirtuCards let alice = Avatar("Alice_VR", "explorer") alice.virtucard = create_virtucard(alice, 1000.0) alice.virtucard.set_security("high")   // Build a virtual marketplace let shop = VirtualTerminal("NexusMarket", "general") shop.list_item("Health Potion", 25.0, "Restores 50 HP")   // Purchase & transfer shop.purchase("Health Potion", alice.virtucard)   let bob = Avatar("Bob_VR", "merchant") bob.virtucard = create_virtucard(bob, 500.0) transfer(alice.virtucard, bob.virtucard, 100.0)   // Legendary item with rarity multiplier let sword = VirtualItem("Dragon Sword", "weapon", 1000.0) sword.set_rarity("legendary") // 10× multiplier print(sword.calculate_value()) // 10000.0
class Player { field health = 100 field name = ""   fn init(player_name) { self.name = player_name }   // Inline method bodies fn heal(amt) { self.health = self.health + amt } fn is_alive() { return self.health > 0 }   fn describe() { print("Player:", self.name, "HP:", self.health) } }   let hero = Player("Nova") hero.heal(50) hero.describe() // Player: Nova HP: 150

VR-Native API

Four powerful built-in types ship with every KursarScript installation.

Avatar Identity
  • Avatar(name, role)
  • .set_attribute(key, value)
  • .get_attribute(key)
  • .get_inventory()

VirtuCard Finance
  • create_virtucard(avatar, amount)
  • .debit(amount)

Installation

Up & Running in Seconds

KursarScript runs on any Python 3.8+ environment with no native dependencies.

1
Install from PyPI
$ pip install kursarscript
2
Or clone from source
$ git clone https://github.com/Infinite-Networker/KursarScript.git
$ cd KursarScript
$ pip install -e .
3
Run a script
$ kursarscript examples/hello_world.kursar
$ kursarscript examples/economy.kursar
4
Launch the interactive REPL
$ kursarscript --interactive
>>> let x = Avatar("Me", "developer")
